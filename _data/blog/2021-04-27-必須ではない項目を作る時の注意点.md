## 必須ではない項目を作る時の注意点

必須ではない項目は、無しにするとフィールド自体が生成されません。\
GatsbyJS を使用している場合、ビルド時にエラーが発生します。\
そのために、空のプレースホルダーを指定しておくとフィールドが生成され、空データとして取得することが可能になります。

```yaml
- { label: 'Title',
    name: 'title',
    widget: 'string' 
    default: 'title'
  }
- {
    label: 'Description',
    name: 'description',
    widget: 'text',
    required: false,
    default: ''
  }
```
